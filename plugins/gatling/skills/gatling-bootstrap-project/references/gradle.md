# Bootstrap Gatling project with Gradle

## Check prerequisites

`java -version` should show at least Java 11.

A Gradle install is not required because the project will include gradlew (Gradle Wrapper).

## Download a sample project

Download the sample project corresponding to the language and build tool, then unzip its content to the target directory:

- Java: https://github.com/gatling/gatling-gradle-plugin-demo-java/archive/refs/heads/main.zip
- Kotlin: https://github.com/gatling/gatling-gradle-plugin-demo-kotlin/archive/refs/heads/main.zip
- Scala: https://github.com/gatling/gatling-gradle-plugin-demo-scala/archive/refs/heads/main.zip

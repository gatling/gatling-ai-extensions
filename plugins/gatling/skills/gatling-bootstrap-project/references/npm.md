# Bootstrap Gatling project with npm

## Check prerequisites

`node --version` should show at least Node.js 20.

## Download a sample project

Download the sample project corresponding to the language and build tool, then unzip its content to the target directory:

- TypeScript + npm: https://github.com/gatling/gatling-js-demo/archive/refs/heads/main.zip and only copy the `typescript` directory from this sample project
- JavaScript + npm: https://github.com/gatling/gatling-js-demo/archive/refs/heads/main.zip and only copy the `javascript` directory from this sample project

## Edit project metadata

Ask user if they want to choose a project name, then edit it in the package.json file. Otherwise keep the default value.
